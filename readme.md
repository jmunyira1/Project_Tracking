# Project Tracking - Academic & Professional Project Management Platform

**A robust platform designed to coordinate and track the progress of academic or professional research projects.**

It facilitates smooth workflow between students and supervisors by breaking projects into measurable **milestones** and
**sub-criteria**, enabling transparent progress monitoring and built-in document version control.

## ✨ Key Features

- **Tiered Milestone Framework**  
  Administrators define project phases (Milestones) and detailed grading criteria (Sub-milestones) with assigned maximum
  marks.

- **Project Registration & Approval Workflow**  
  Formal process: students register project titles & descriptions → supervisor review → administrative approval.

- **Visual Progress Dashboard**  
  Real-time progress bars showing:  
  • Individual student completion percentages  
  • Overall milestone submission rates across the system

- **Document Versioning System**  
  Automated version tracking for student submissions. Supervisors can view full upload history per milestone.

- **Stakeholder Management**  
  Dedicated modules for student and supervisor profiles, including:  
  • Registration numbers  
  • Course affiliations  
  • Contact information

- **Interactive Evaluation Tools**  
  Clean UI using Bootstrap accordions and modals for managing complex criteria lists and document previews.
